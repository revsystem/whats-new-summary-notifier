Slackに通知を受信するチャンネルを作成する｡
api.slack.com でIncoming Webhooksを開き、Webhook URLを取得する｡

Sample curl request to post to a channel のcurlコマンドを実行し､Slackに通知が送信されることを確認する｡

Webhook URLをAWS Systems Managerのパラメータストアに保存する｡
```
aws ssm put-parameter \
  --name "/WhatsNew/URL" \
  --type "SecureString" \
  --value "<Input your Webhook URL>"
```

aws-samples からリポジトリをクローンする
```
git clone https://github.com/aws-samples/whats-new-summary-notifier.git
```

CDKのデプロイを行う
```
cd whats-new-summary-notifier
npm ci
cdk bootstrap   # 既に CDK を利用したことがある方は不要です。
 ⏳  Bootstrapping environment aws://531713114752/us-east-1...
Trusted accounts for deployment: (none)
Trusted accounts for lookup: (none)
Using default execution policy of 'arn:aws:iam::aws:policy/AdministratorAccess'. Pass '--cloudformation-execution-policies' to customize.
CDKToolkit: creating CloudFormation changeset...
 ✅  Environment aws://531713114752/us-east-1 bootstrapped.


cdk deploy
```

npmコマンドがない場合はインストールする
```
sudo apt install npm
```

CDKがない場合はインストールする
```
sudo npm install -g aws-cdk
```

cdk deployでError: spawnSync docker ENOENTが発生する場合は、dockerをインストールする｡
